# 🍔 Food Delivery App – UI/UX Design

## 📌 Project Overview
This project is a modern **Food Delivery Mobile Application UI/UX Design** created in Figma.  
The app focuses on providing a seamless food ordering experience with intuitive navigation, clean layout, and user-friendly interactions.

## 🎯 Project Goals
- Simple and smooth food ordering experience
- Clean and modern UI design
- Easy restaurant browsing
- Fast checkout process
- User profile & order tracking system
